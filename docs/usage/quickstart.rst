Quickstart
==========


TL;DR
-----

.. code-block:: bash

    $ python3 -m venv venv --prompt rune
    $ source venv/bin/activate
    (rune) $ pip install -U pip wheel rune
    (rune) $ flask core init -d
    (rune) $ flask db init
    (rune) $ flask db migrate
    (rune) $ flask db upgrade
    (rune) $ flask auth create-admins
    (rune) $ flask run


http://localhost:5000

Username: rune

Password: RUNE
